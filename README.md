# ♻️ Waste Management System

A web-based Waste Management System developed to manage waste collection, pickup requests, monitoring, and disposal efficiently.

## Features

* User Registration & Login
* Waste Pickup Request
* Waste Collection Management
* Complaint & Feedback System
* Admin Dashboard
* Database Management

## Tech Stack

* Java
* JSP & Servlet
* Oracle Database
* Apache Tomcat
* NetBeans IDE

## Installation

1. Clone the repository
2. Open project in NetBeans
3. Configure Oracle Database
4. Run using Apache Tomcat Server
